About Rust-IOCP

Rust-IOCP is a Windows input output completion port library for in Rust.

Rust-IOCP binds to the Windows API and provides support for the IoCompletionPort.

The project can be found on GitHub and on crates.io.

Installation

To add a library release version from crates.io to a Cargo project, add this to the [dependencies] section of your Cargo.toml:

iocp = "~0.0.5"

To add the library's Git repository to a Cargo project, add this to your Cargo.toml:

[dependencies.iocp]

git = "https://github.com/cyderize/rust-iocp.git"

And add the line extern crate iocp; to your main.rs or lib.rs.
